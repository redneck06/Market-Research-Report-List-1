<p><p>In the market for in-display fingerprint sensors, there are three main types available. The first one is the optical in-display fingerprint sensor, which uses light to capture an image of the fingerprint. The second type is the ultrasonic in-display fingerprint sensor, which uses sound waves to create a 3D map of the fingerprint. Lastly, there is the capacitive in-display fingerprint sensor, which measures the electrical signals in the fingerprint to authenticate the user. These different types offer various levels of security and accuracy for unlocking devices.</p></p>
