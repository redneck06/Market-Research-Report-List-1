<p><p>The multi yogurt machine market consists of two main types: full-automatic and semi-automatic. The full-automatic type requires minimal user intervention as it automatically completes the entire yogurt-making process, including mixing, fermenting, and cooling. On the other hand, the semi-automatic type requires the user to perform certain steps manually, such as adjusting the temperature or monitoring the fermentation time. Both types cater to different consumer needs, with the full-automatic type offering convenience and ease of use, while the semi-automatic type allows for more customization and control during the yogurt-making process.</p></p>
